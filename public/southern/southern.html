Angular Exercise

Create an angular app that allows a user to add a new email address. The email address should be validated. If it is not valid present the user with an error message. After successful validation, add the new email address as an option in a select element.

    Of note:
  • UI presentation matters. Use Bootstrap to style the UI elements.
  • We cannot receive zip files due to IT security. Please host it somewhere or create a CodePen.

Enter a valid email. ex. john.doe@example.com
